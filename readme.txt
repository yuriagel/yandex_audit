Задание
Написать библиотеку для сбора статистики по неким событиям
Например, клиент может встроить библиотеку для мониторинга публикации соообщений в новостной ленте.
Библиотека предоставляет клиентам класс с интерфейсом
1. учесть событие. на входе время события.
2. Выдать число событий за последнюю минуту. Нет входных данных
3. Выдать число событий за последний час. Нет входных данных
4. Выдать число событий за последние сутки. Нет входных данных

События поступают в систему асинхронно  в произвольный момент времени.
Возможна нагрузка как 10000 событий в секунду, так и 2-х в час.
Библиотека не предоставляет клиентам функциональность постоянного хранения событий и статистики по событиям.
